## ğŸ’» Getting Started <a name="getting-started"></a>

### Setup

To setup the project locally: clone the repository using:

```
git clone https://github.com/SimbaWANGU/hello-rails-back-end.git
```

### Install dependencies
run:
```
bundle install
```

### Setup database
run:
```
rails db:create
rails db:migrate
rails db:seed
```

### Run the app
run:
```
rails s
```
